# Document Management

The Document Management domain provides file and folder management capabilities.

## What It Provides

- File, file-content, folder, and folder-role services.
- Document APIs used by the aggregate web host.
- WebDAV-style middleware coverage for document operations.
- Baseline document/content assets used during platform setup.

## Why It Matters

Consumers get a platform-level document area that can be secured, exposed through APIs, and used by other platform features such as content and workflow.
