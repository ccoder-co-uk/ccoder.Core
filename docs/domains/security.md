# Security

The Security domain provides the platform security and SSO persistence foundation.

## What It Provides

- Tenant, user, role, privilege, and user-role services.
- Security database context abstractions and SQL Server-backed persistence.
- Shared security infrastructure used by application-level authorization and platform setup.

## Why It Matters

Security is a foundation dependency for the aggregate platform. It supplies the lower-level identity and authorization data used by the higher-level App Security domain.
