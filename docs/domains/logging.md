# Logging

The Logging domain provides platform log capture and streaming support.

## What It Provides

- Log entry and log data item storage services.
- Authorization-aware log APIs.
- SignalR-backed logging exposure for web and hosted-service diagnostics.
- Acceptance coverage for log hub negotiation and console log sending.

## Why It Matters

Consumers get a common logging surface across the aggregate platform, rather than each domain inventing separate diagnostics storage and streaming behavior.
