# App Security

The App Security domain models permissions within cCoder applications.

## What It Provides

- Application, role, privilege, user, and user-role orchestration.
- Event-aware services for adding, updating, and deleting security-related application data.
- Hosted-service registration support for app security bootstrap and app-add processing.

## Why It Matters

Consumers get application-level access control on top of the shared security model, so apps can manage their own roles and privileges while still using the common platform identity layer.
