# Mail

The Mail domain provides email configuration, queueing, and sent-email tracking.

## What It Provides

- Mail server management.
- Queued email and sent email services.
- Mail-related API exposure through the aggregate host.
- Integration points for workflow and user-registration scenarios that need email delivery.

## Why It Matters

Consumers get a platform mail subsystem that can be used by security, workflow, notification, and application features without building mail queue/storage behavior separately.
