# Workflow

The Workflow domain provides workflow definitions, activities, and execution support.

## What It Provides

- Workflow definition and workflow event services.
- Shared workflow activities through `cCoder.Workflow.Activities`.
- Hosted-services integration for background workflow execution.
- Integration-test coverage across Web, HostedServices, and Workflow process boundaries.

## Why It Matters

Consumers can model business processes as platform workflows and run them through the aggregate hosted-services architecture.
