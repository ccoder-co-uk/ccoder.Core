# Content Management

The Content Management domain provides metadata-driven content and UI asset management for the platform.

## What It Provides

- Resource, component, script, layout, template, and page-related platform content.
- Support for baseline assets and latest exported content assets used by acceptance tests and setup flows.
- Services that allow applications to manage configurable content separately from compiled application code.

## Why It Matters

Consumers can build applications where UI resources and content definitions are platform data rather than hard-coded application behavior.
